Infanalyze:
一种用于婴幼儿大脑结构磁共振影像数据分析的多功能平台

一、软件介绍

Infanalyze是一款深度融合医学图像处理技术、具有医学诊断功能的专业级工具箱，由南京航空航天大学计算机科学与技术学院、人工智能学院学生团队开发。该软件基于MATLAB平台开发，专注于婴幼儿大脑影像数据分析领域，致力于为非影像学专业背景的研究人员提供一套高效、精确、操作便捷的大脑影像分析工具。

该平台功能完备，全面涵盖了结构磁共振影像（Magnetic Resonance Imaging, 简称MRI）数据预处理、后期分析和结果可视化等多个关键流程。软件依托开源的FreeSurfer库和多个Python脚本，通过对各函数进行优化改进和深度整合，实现了对婴幼儿大脑MRI数据的全自动、多步骤端到端分析。具体到预处理流程，软件能够高效有序地完成图像格式转换、图像标准化、头骨剥离、组织分割、皮层重建和配准等一系列操作，有效去除图像中的噪声干扰，统一图像格式与标准，精准提取脑部组织信息；在后期分析阶段，软件支持从皮层水平和体素水平等多个维度提取多种大脑结构和形态学特征。软件能够自动将处理结果整合成标准的PDF报告，便于研究人员查验和使用。此外，软件还具备动态脑部影像的可视化功能，支持医生和研究人员从多个视角观察和分析脑部结构，为临床诊断和科研工作提供坚实可靠的依据。

二、安装方法

软件运行的推荐电脑配置：请确保最低配置内存是32G的运行内存

为保证软件运行稳定，减少使用过程中因软件更新引发的版本适配问题，建议用户使用MATLAB R2022b版本。以下是软件的具体安装步骤：

  • 前往Docker官网下载并安装Docker Desktop软件(Docker Desktop安装使用教程)。启动Docker Desktop，打开本机终端（可选择cmd或Powershell），输入指令

    docker pull crpi-871vgyf837zm6nsl.cn-shanghai.personal.cr.aliyuncs.com/matlab_c/matlabapp:1.0

    拉取镜像，在此期间，请务必确认您的电脑已成功安装DockerDesktop且该软件处于运行状态。指令执行成功后，系统会从指定的阿里云仓库地址下载 matlabapp:1.0 镜像至您的本地Docker镜像库中。需留意的是，在拉取镜像到本地Docker 镜像库时，默认的下载存储路径为本机C盘，因此建议您确保C盘已预留足够的内存空间（至少50G，镜像实际大小为39G）如果需要修改储存路径，可以在Docker的setting界面中点击Resources，设置docker image的位置为目标路径。
  • 拉取镜像成功后，再输入指令

    docker run -it --name matlab_c crpi-871vgyf837zm6nsl.cn-shanghai.personal.cr.aliyuncs.com/matlab_c/matlabapp:1.0 /bin/bash

    来运行镜像，创建容器实例。
  • 从百度网盘链接下载软件运行所需的代码文件，保存在和被试同目录下的文件夹中。 【下载软件】

四、结果展示

特征提取模块支持用户基于预处理后的影像数据，按需导出不同模态的特征数据集，实现多维度可视化观察与定量化统计分析。特征提取功能按数据结构划分为以下两类：

  • Surface特征提取
    用户需预先指定脑区分割方案（DKT-40分区或Lobe脑叶分区）作为特征计算的空间基准。支持提取以下四类皮层形态学指标：大脑表面积、大脑皮层厚度、曲率指标（包括平均曲率，高斯曲率，曲率系数和折叠系数四种）以及大脑皮层灰质体积。
  • Volume特征提取
    用户可直接勾选目标脑区体积，包括：总脑体积、脑干体积、小脑体积、大脑皮层下区域体积、胼胝体体积（anterior，mid anterior， central，posterior，mid posterior部分）以及脑室体积。

以下以Surface部分选择DKT-40且选择其中的皮层表面积、皮层厚度这两个选项，Volume 部分选择脑室体积这一选项为例，详细说明软件的运行及数据产出情况。

Surface部分执行上述选择后，软件会生成一个DKT-Surface文件夹和两个xlsx表格（DKT-lh.xlsx和DKT-rh.xlsx）：

  • DKT-Surface文件夹在运行过程中仅会产生一些过程性文件，该文件夹内包含分别存放左右脑结果的lh和rh文件夹，各文件夹下包含对应选定特征的数据统计表，例如SurfArea.csv文件记录皮层表面积数据，ThickAvg.csv文件记录皮层厚度数据，各特征的所有数据均存储在后缀为.csv表格中。在运行完成后，DKT-Surface文件夹会自动清空，以避免对下次运行造成数据污染。
  • 两个xlsx表格（DKT-lh.xlsx和DKT-rh.xlsx）包含是最终的结果数据，表格具体呈现了各分割区域（如楔叶、内嗅回、纺锤体等等）对应选定特征的数据：
